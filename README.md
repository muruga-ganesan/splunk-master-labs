# Splunk Master Labs

## Overview

This repository contains my hands-on Splunk learning journey, including practical labs, SPL (Search Processing Language) queries, dashboards, alerts, reports, and production troubleshooting scenarios.

The objective is to gain real-world experience in log analysis, monitoring, observability, and incident investigation for Production Support and Site Reliability Engineering (SRE).
